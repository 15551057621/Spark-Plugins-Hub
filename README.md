# Spark-Plugins-Hub
这是一个存储SparkBridge3框架插件的仓库

## 插件列表

### DeepSeek - 在你的QQ群内接入DS酱吧~ v0.0.3
- **加载模式**: bds
- **优先级**: post
- **权限**: key  
一个简单的插件，将DeepSeek接入Q群  
自用插件，有bug可以提，可二创  
支持显示token调用量


### QQChatEx - 更好的聊天转发插件 v1.0.0
- **加载模式**: bds
- **优先级**: post
- **权限**: core  
依旧是自用插件，加了个配置发出来了（

#### 它有什么功能？
- 敏感词过滤（需要前置插件）
怕玩家发了某些敏感词导致号被封？  
没有关系！可以用这个功能过滤  

- Q群内回复时自动在mc内at
比如我想要对某个玩家回复，只需要把之前的消息拉在引用里  
然后发送就行了，超级超级便捷！  

- 自动获取群昵称
sb3自带的那个不会自动转换名称，只会发个QQ号  
这个插件可以自动转换群昵称  

- 控制台聊天转发
sb3自带的那个插件不会在控制台转发  
虽然没啥吧，但是看着挺不舒服的  
这个插件会转发到控制台  

- 命令执行
这个就不用多说了吧，跟原版执行方式一样  
斜杠开头视作命令执行，并将执行结果输出到控制台  

### QQDebug - 在QQ群内在线调试吧~ v0.0.1
- **加载模式**: bds
- **优先级**: post
- **权限**: core  
如果你觉得在sb3插件开发时，反复重载/敲一大串jsdebug太麻烦的话，可以试试这个插件！  

此插件适用于sb3开发者，如果你是普通用户，可以下来玩玩，但一定要处理好权限关系  
插件使用core权限，请注意好隐私管理！  
依旧自用插件，欢迎PR和反馈建议owo  

#### 命令参数
- debugdata 查看调试信息结构
- reply 查看引用的消息元数据
- msgdata 查看当前群聊信息
- sendmsg 发送CQ码消息
  - 此参数还有以下子参数
  - "file2base64" - 把${}包裹的路径解析为base64
  - "msgdata" - 查看这条CQ码消息的解析

### QQGetLogs - 在QQ群内在线获取日志吧~ v0.0.1
- **加载模式**: bds
- **优先级**: post
- **权限**: core  
让玩家可以在QQ群里自助获取日志！不需要手动登录后台下载发送！  
支持自定义日志文件目录、命令、可以使用命令的群聊！  

### SimpleLike - 在你的QQ群内使用点赞功能吧~ v0.0.1
- **加载模式**: bds
- **优先级**: post
- **权限**: key  
一个超级简单的点赞插件  
发送 /赞我 获取点赞

### SkyHelp - 在QQ群内快速获取光遇信息吧～支持自定义api！ v1.0.0
- **加载模式**: bds
- **权限**: key
- **优先级**: main

## 使用说明
每个插件都包含相应的配置文件(spark.json)和实现文件(index.js)，请按照SparkBridge3框架的要求进行安装和配置。
